# YAML

> YAML 是一种非常简洁、强大、专门用来编写配置文件的语言
> 全称: YAML Ain't Markup Language
> 该语言的设计参考了JSON/XML/SDL等语言
> 已数据为中心，简洁易读，编写简单

## 语法特点

* 大小写敏感
* 通过缩进表示层级关系
* 禁止使用tab缩进，只能使用控制
* 所进的空格数量不重要，只要相同层级左对齐
* 使用#表示注释
